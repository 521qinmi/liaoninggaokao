## 安装与运行说明

### 系统要求
- Node.js 16 或更高版本
- PostgreSQL 12 或更高版本
- 任何现代浏览器

### 快速开始（3 步）
1. **初始化数据库**
   ```bash
   createdb college_choice
   psql -U postgres -d college_choice -f backend/src/db/schema.sql
   ```

2. **启动后端**
   ```bash
   cd backend && npm install && npm run dev
   ```

3. **启动前端**
   ```bash
   cd frontend && npm install && npm run dev
   ```

访问 `http://localhost:5173` 即可
